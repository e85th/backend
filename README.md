# backend

Currently provides only the basics for a backend server.  Basics include:
    * Store users and passwords
    * Associate contact information ("channels")
    * Log events

## Installation

    * Run ```db/ddl.sql```
    * Run ```db/seed.sql```

## Usage

    $ java -jar backend-0.1.0-standalone.jar [args]
